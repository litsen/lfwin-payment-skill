# 条码支付（B扫C）

> 商户使用扫码枪扫描顾客手机出示的付款码完成收款。系统可自动识别微信/支付宝/云闪付，也可手动指定通道。

---

## 1 聚合条码支付（推荐，自动识别）

> 使用 `pay.comm.barcode`，系统自动判断付款码类型（微信/支付宝/云闪付），无需商户事先区分。

**请求地址**：`POST /payapi/pay/barcode`  
**service**：`pay.comm.barcode`

### 请求参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| service | string | 是 | 接口名称，固定值：`pay.comm.barcode` |
| apikey | string | 是 | 商户 APIKEY |
| money | string | 是 | 支付金额，单位：元，最小 0.01 |
| dynamic_id | string | 是 | 顾客付款码（条码设备扫描获取） |
| nonce_str | string | 是 | 随机字符串（≤32位） |
| sign | string | 是 | 签名 |
| mch_orderid | string | 否 | 商户订单号（唯一） |
| fq_num | string | 否 | 分期期数 |
| seller_percent | string | 否 | 商家承担分期手续费比例（%） |
| goods_tag | string | 否 | 商品标签 |
| mid | string | 否 | 商户 MID |
| remarks | string | 否 | 订单备注 |
| guid | string | 否 | 全局唯一标识 |
| store_dis | string | 否 | 平台店铺优惠活动开关：`1` 打开，`0` 关闭 |
| limitpay | string | 否 | 限制支付 |
| alipay_store_id | string | 否 | 支付宝门店ID |
| time_expire | string | 否 | 订单失效时间（时间戳秒），最长2小时 |
| sence_no | string | 否 | 场景编号 |
| goods_info | string | 否 | 商品详情，JSON字符串格式 |
| term_info | string | 否 | 终端信息 |
| sign_type | string | 否 | 签名方式：MD5/RSA，默认 MD5 |
| split_info | string | 否 | 动态分账规则（JSON格式） |
| scene | string | 否 | 支付场景 |
| terminal_params | string | 否 | 终端参数 |
| good_name | string | 否 | 商品名称 |

### 请求示例

```
service=pay.comm.barcode&apikey=00014005&money=0.01&dynamic_id=286103991909766417&nonce_str=123456&sign=1274888ff40215620e4141a5392c49c6
```

### 返回参数

| 参数名 | 描述 | 类型 |
|--------|------|------|
| service | 接口名称 | string |
| orderid | 平台订单号 | string |
| trade_no | 通道流水号 | string |
| paymoney | 订单金额（单位：元） | string |
| pri_paymoney | 商户实收金额 | string |
| receipt_amount | 渠道实收金额 | string |
| buyer_pay_amount | 用户支付金额 | string |
| point_amount | 积分支付金额 | string |
| coupon_fee | 优惠券支付金额 | string |
| buyer_account | 买家账号 | string |
| paytime | 支付时间（时间戳） | string |
| fund_bill_list | 渠道支付工具明细（JSON） | string |
| version | 接口版本号：3.0 | number |
| charset | 字符编码：UTF-8 | string |
| message | 错误信息 | string |
| status | 错误状态 | number |
| sign_type | 签名方式 | string |
| sign | 签名字符串 | string |

### 返回示例

```json
{
    "service": "pay.comm.barcode",
    "orderid": "20180202171044007002473",
    "trade_no": "4200000061201802025681296197",
    "paymoney": "0.01",
    "pri_paymoney": "0.01",
    "receipt_amount": "0.01",
    "buyer_pay_amount": "0.01",
    "point_amount": "0.00",
    "coupon_fee": "0.00",
    "buyer_account": "微信账号",
    "paytime": "1517562682",
    "fund_bill_list": "[]",
    "status": "10000",
    "message": "SUCCESS"
}
```

### 付款码格式参考

| 支付方式 | 开头 | 位数 |
|----------|------|------|
| 微信 | `10`、`11`、`12` | 18位纯数字 |
| 支付宝 | `25` | 16~24位 |
| 云闪付 | `62` | 19位 |

## 2 指定通道条码支付

> 若需明确指定某一通道，可使用对应的 service 值，覆盖场景与聚合条码支付完全一致。

| 支付方式 | service 值 |
|----------|-----------|
| 支付宝 | `pay.alipay.barcode` |
| 微信支付 | `pay.wxpay.barcode` |
| 云闪付 | `pay.unpay.barcode` |

**请求参数、返回参数均与聚合条码支付完全相同**，仅 `service` 值不同，不再单独列出。

## 3 撤销订单

> 本接口针对刷卡支付（被扫）模式，用户长时间未等到订单终态或者一直超时，调用本接口撤销订单。当银行认定该订单已支付成功或支付失败，则撤销失败。调用订单撤销前请先进行订单查询。
> 最早可撤销时间在订单创建之后15秒后。
> 撤销交易需要控制在1:00—23:00，其余时间不允许进行撤销交易。

**请求地址**：`POST /payapi/pay/cancel_order`  
**service**：`pay.comm.cancel_order`

### 请求参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| service | string | 是 | 接口名称，固定值：`pay.comm.cancel_order` |
| apikey | string | 是 | 商户 APIKEY |
| orderid | string | 是 | 系统订单号，与商户订单号mch_orderid、dis_name三选一 |
| nonce_str | string | 是 | 随机字符串（≤32位） |
| sign | string | 是 | 签名 |
| mch_orderid | string | 否 | 商户订单号，与orderid二选一 |
| order_time | string | 否 | 下单时间（时间戳秒），只传mch_orderid时必传 |
| dis_name | string | 否 | 间联渠道订单号 |
| reason | string | 否 | 撤销理由 |
| sign_type | string | 否 | 签名方式：MD5/RSA，默认 MD5 |

### 请求示例

```
service=pay.comm.cancel_order&apikey=00014005&orderid=20180202171044007002473&nonce_str=123456&sign=d347ea0a78d2e0581aecca6634e2748d
```

### 返回参数

| 参数名 | 描述 | 类型 |
|--------|------|------|
| service | 接口名称 | string |
| orderid | 系统订单号 | string |
| mch_orderid | 商户订单号 | string |
| status | 状态码 | string |
| message | 提示信息 | string |
| version | 接口版本号：3.0 | string |
| charset | 字符编码：UTF-8 | string |
| sign_type | 签名方式 | string |
| sign | 签名字符串 | string |

### 返回示例

```json
{
    "service": "pay.comm.cancel_order",
    "orderid": "20180202171044007002473",
    "mch_orderid": "",
    "status": "10000",
    "message": "SUCCESS"
}
```

### 注意事项
- 撤销订单后，支付金额原路退回顾客账户
