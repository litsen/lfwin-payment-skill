## 统一收银台

> 调用该接口可以快速对接支付，无需对接多个支付方式，直接调起支付。

- **请求地址**：`/index/Payment/pre_order`
- **请求方式**：POST
- **调试方式**：可以使用该工具进行调试收银台链接获取方式 `https://api2.lfwin.com/index/Payment/preorder`

## 请求参数

| 参数名 | 类型 | 必填 | 描述 | 示例 |
|--------|------|------|------|------|
| apikey | string | 是 | 商家APIKEY，唯一标识，由服务商提供 | `00014005` |
| money | string | 是 | 支付金额，单位元，最小值0.01 | `0.02` |
| mch_orderid | string | 是 | 商户订单号（正常需在商户系统上唯一） | `TS123123123123` |
| sign | string | 是 | 签名，参照签名算法 | `470b767963acbcf45cf6aeb8f2b1bc4d` |
| nonce_str | string | 是 | 随机字符串，长度不超过32位 | `1001190599123456` |
| succ_url | string | 否 | 支付成功回跳地址（部分支付方式支持） | |
| err_url | string | 否 | 支付失败回跳地址（部分支付方式支持） | |
| notify_url | string | 否 | 支付成功状态通知地址 | |
| goods_tag | string | 否 | 商品标签 | |
| goods_info | string | 否 | 商品详情，单品优惠活动该字段必传，JSON字符串格式 | |
| split_info | string | 否 | 动态分账规则：json格式，account：商户号；amount：分账金额，单位元 | |
| split | string | 否 | 分账标识（乐刷） | |
| time_expire | string | 否 | 订单超时时间 | |

## 请求示例

```
apikey=00014005&money=0.02&mch_orderid=TS123123123123&nonce_str=1001190599123456&sign=470b767963acbcf45cf6aeb8f2b1bc4d&sign_type=MD5
```

## 响应参数

| 参数名 | 类型 | 描述 |
|--------|------|------|
| data | string | 支付跳转URL或支付参数 |
| version | string | 接口版本号，3.0 |
| charset | string | 接口字符编码，UTF-8 |
| message | string | 错误信息 |
| status | string | 错误状态，10000表示成功 |
| sign_type | string | 签名方式，MD5 |
| sign | string | 签名字符串 |
